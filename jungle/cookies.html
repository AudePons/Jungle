Cookies.

Généralités :
Les cookies sont de petits fichiers texte stockés dans votre navigateur lorsque vous utilisez des sites Web. Vous pouvez contrôler la manière dont les sites Web utilisent les cookies en configurant les paramètres de confidentialité de votre navigateur (consultez l’aide de votre navigateur pour en savoir plus sur le contrôle des cookies). Remarque : Si vous désactivez complètement les cookies, les sites Web Jungle risquent de ne pas fonctionner correctement.

Jungle et les entreprises qui prennent part à son activité utilisent les cookies de diverses manières, par exemple :
• Ils vous authentifient et vous identifient sur nos sites Web afin que nous puissions fournir les services demandés.
• Ils assurent le suivi des informations que vous nous avez fournies. Par exemple, ils conservent les articles dans votre panier lorsque vous visitez Jungle.com.
• Ils permettent de vous proposer les sites Web Jungle que vous utilisez.
• Ils mémorisent vos préférences ou l’endroit où vous étiez lorsque vous avez cessé d’utiliser un site Web Jungle.
• Ils mesurent votre usage des sites Web Jungle afin que nous puissions améliorer ces derniers, adapter nos sites Web à vos centres d’intérêt potentiels, ainsi que réaliser des études de marché.
• Ils analysent vos centres d’intérêt potentiels, ce qui permet à Jungle de proposer des publicités et un contenu plus pertinents sur les sites Web et applications qui n’appartiennent pas à Jungle.
• Ils exécutent les solutions Jungle Experience Cloud qui aident nos clients professionnels à personnaliser et à améliorer les performances de leurs sites Web, applications et messages marketing.
Technologies similaires aux cookies :
D’un point de vue technique, les cookies sont appelés des « cookies HTTP ». D’autres technologies, comme le stockage local HTML5 et les objets partagés locaux (LSO), servent les mêmes objectifs. Les LSO sont utilisés par les auteurs de fichiers lus par Jungle ® Flash® Player et les sites Web qui les hébergent (en savoir plus sur Flash Player et les LSO). Nous sommes parfois amenés à utiliser le stockage local HTML5, les LSO et des technologies similaires pour vous authentifier, assurer le suivi des informations que vous nous avez fournies et mémoriser vos préférences (voir les points ci-dessus). Lorsque vous utilisez une application Jungle hors ligne, nous pouvons stocker les informations liées à la manière dont vous avez utilisé ce site Web sur votre appareil et les transférer ensuite à nos serveurs la prochaine fois que vous vous connecterez en ligne à notre service.
Les pixels invisibles et scripts intégrés :
Les pixels invisibles et scripts intégrés sont d’autres technologies qu’ Jungle utilise sur ses sites Web, ainsi que dans certains e-mails et publicités.
Les pixels invisibles (ou « balises ») sont de petits codes de programmation inclus dans les pages Web, e-mails et publicités qui avertissent Jungle (ou les entreprises qui prennent part à son activité) lorsqu’un utilisateur consulte ces pages Web, e-mails ou publicités ou clique dessus.
Les scripts intégrés sont de petits codes de programmation inclus dans certaines de nos pages Web qui mesurent votre comportement sur ces pages Web, par exemple les liens sur lesquels vous cliquez. Ces informations nous sont utiles pour améliorer nos sites Web, les adapter à vos centres d’intérêt potentiels et réaliser des études de marché. Vous pouvez désactiver la fonctionnalité de script, par exemple JavaScript, dans votre navigateur (consultez l’aide de votre navigateur). Remarque : si vous désactivez la fonctionnalité de script, certains sites Web Jungle risquent de ne pas fonctionner correctement.
